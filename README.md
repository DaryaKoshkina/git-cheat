# Git — 20 главных команд

| Команда | Что делает |
|---------|------------|
| `git init` | Создать репозиторий в текущей папке |
| `git clone <url>` | Скачать удалённый репозиторий |
| `git status` | Показать статус (что изменено/не добавлено) |
| `git add .` | Добавить все изменения в staging |
| `git commit -m "сообщение"` | Сделать коммит (сохранить snapshot) |
| `git log --oneline` | Показать историю коммитов (сокращённо) |
| `git branch` | Список локальных веток |
| `git checkout -b <ветка>` | Создать новую ветку и переключиться на неё |
| `git checkout <ветка>` | Переключиться на другую ветку |
| `git merge <ветка>` | Слить указанную ветку в текущую |
| `git branch -d <ветка>` | Удалить ветку |
| `git remote add origin <url>` | Привязать удалённый репозиторий |
| `git push origin main` | Отправить изменения на удалённый репозиторий |
| `git pull origin main` | Забрать изменения с удалённого репозитория |
| `git fetch origin` | Скачать изменения без слияния |
| `git diff` | Показать различия (до `git add`) |
| `git restore <файл>` | Отменить изменения в файле |
| `git reset --hard HEAD~1` | Отменить последний коммит (удалить изменения) |
| `git stash` | Временно спрятать незакоммиченные изменения |
| `git stash pop` | Вернуть спрятанные изменения |# git-cheat
